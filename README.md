# Yojana Guide - Welfare Schemes Information Portal

Yojana Guide is a comprehensive, high-depth information portal detailing over 50 Indian government welfare schemes and programs. Built with modern web standards (pure HTML, vanilla CSS, and lightweight JS), it is fully optimized for speed, search engines (SEO), YMYL (Your Money Your Life) compliance, and Google AdSense guidelines.

Live Site: [https://yojana-three.vercel.app](https://yojana-three.vercel.app)

---

## 🌟 Key Features

* **50+ High-Depth Scheme Guides:** Comprehensive details on benefits, eligibility criteria, required documents, application process, and status checks.
* **12 Welfare Categories:** Quick filtering across sectors including Agriculture, Health, Housing, Education, Social Security, Pension, and Financial Inclusion.
* **AdSense & YMYL Compliance:**
  * **E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness):** Verifiable editorial profile (Senior Policy Analyst & Welfare Specialist) with checkmark badges and contact paths embedded on all article pages.
  * **Sourced Statistics:** Clear, defensible statistics (e.g. 11.8 Cr+ PM-Kisan Farmers, 34.5 Cr+ Ayushman Cards) citing official government portal records.
  * **Compliance Pages:** Clear and accessible Privacy Policy, Disclaimer, Terms & Conditions, and About/Contact sections.
* **Modern Premium UI/UX:**
  * Vibrant CSS gradients, sleek typography (Outfit & Inter), and polished glassmorphism elements.
  * 100% responsive navigation dropdowns, mobile-friendly hamburger menu overlay, and back-to-top indicators.
  * Reading progress bar indicator on all article detail pages.
* **SEO Optimization:**
  * Standardized canonical URLs targeting `yojana-three.vercel.app` to prevent search indexing duplication.
  * Dynamic, clean JSON-LD structured data schema (WebSite and Article types) on all pages.
  * Automatically generated XML Sitemap (`sitemap.xml`) and `robots.txt` configuration.

## 🛠️ Development & Compilation Workflow

The project uses a lightweight Python compiler to keep static files synchronized, preventing manual edit mismatches.

### 1. Rebuild Scheme Pages
If you make changes to core scheme structures, content text, tables, or FAQs in `generate_schemes.py` or the facts dictionary in `rebuild_high_depth_pages.py`, compile them using:
```bash
python rebuild_high_depth_pages.py
```
This generates/overwrites the 50 scheme `.html` files in the root folder.

### 2. Synchronize Navigation & Generate Sitemap
To propagate updates in header links, footer contents, or popular scheme dropdown items to all 57 pages of the site, run the synchronization script:
```bash
python update_all_pages.py
```
This will:
* Synchronize navigation menus site-wide (aligning titles like `PM Kisan Samman Nidhi` and `Ayushman Bharat PM-JAY`).
* Inject Google AdSense scripts into the `<head>` of all files.
* Regenerate the `sitemap.xml` with the correct last modified dates and priority headers.
